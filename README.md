# WoW Export

An attempt at creating an universal WoW Exporter

## Disclaimer

This project is based on [Marlamin's](https://github.com/Marlamin) tools

## Compatibility table

| | ADT | M2 | WMO |
| --- | --- | --- | --- |
| Vanilla | No | No | Partial |
| TBC | No | No | Partial |
| WOTLK | Yes | Yes | Yes |
| CATA | Yes | Yes | Yes |
| MOP | Yes | Yes | Yes |
| WOD | Yes | Yes | Yes |
| Legion | Yes | Yes | Yes |
| BFA | No | No | No |
| Classic | No | No | No |
